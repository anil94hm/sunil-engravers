HOW TO ORGANIZE YOUR PHOTOS INTO CATEGORIES
====================================================================

STEP 1: Understand the Folder Structure
----------------------------------------
Your images folder now has these category folders:

C:\Users\Anil Halebeedu\sunil-engravers-website\images\
├── rings/        (for ring photos)
├── earrings/     (for earring photos - tops, studs, etc.)

├── bangles/      (for bangle and bracelet photos)
├── pendants/     (for pendant photos)
└── other/        (for any other jewelry items)

STEP 2: Move Photos to Category Folders
----------------------------------------
1. Look at each photo (1.jpg, 2.jpg, 3.jpg, etc.)
2. Identify what type of jewelry it is
3. COPY (not move) the photo to the appropriate category folder
4. Rename it with a number in that folder

EXAMPLE:
--------
Original photo: images/15.jpg (shows a ring)
Action: Copy it to: images/rings/1.jpg

Original photo: images/42.jpg (shows earrings)
Action: Copy it to: images/earrings/1.jpg

Original photo: images/73.jpg (shows a necklace)
Action: Copy it to: images/necklaces/1.jpg

DETAILED INSTRUCTIONS
====================================================================

Method 1: Manual Organization (Recommended)
-------------------------------------------
1. Open two File Explorer windows side by side
2. In first window: Navigate to images folder
3. In second window: Navigate to category folder (e.g., images/rings/)
4. Look at each numbered photo
5. Copy photos to appropriate category folders
6. Rename them sequentially in each folder (1.jpg, 2.jpg, 3.jpg...)

Example workflow:
- images/5.jpg is a ring → copy to images/rings/1.jpg
- images/8.jpg is also a ring → copy to images/rings/2.jpg
- images/12.jpg is an earring → copy to images/earrings/1.jpg
- images/20.jpg is a necklace → copy to images/necklaces/1.jpg

Method 2: Quick Batch Organize
------------------------------
1. Create a temporary folder for each category
2. Go through all 100 photos
3. Copy each photo to its category temp folder
4. Then copy all from temp folders to category folders with numbering
